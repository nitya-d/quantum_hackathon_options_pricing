# Track A: Hardware Challenge - Preparation Guide

**Sponsor:** Rigetti Computing

## Focus Area

This track will focus on **Optimization** using quantum-classical hybrid approaches. Participants should prepare by understanding QAOA (Quantum Approximate Optimization Algorithm).

## Required Tools (Open Source)

- **Python Libraries:** `numpy`, `scipy`, `networkx`
- **Quantum SDKs:** `pyQuil`, `Qiskit`, or `Cirq`
- **Rigetti SDK:** [Forest SDK & QCS](https://docs.rigetti.com/qcs)

## Study Resources (Recommended Reading)

To succeed in this challenge, we recommend reviewing:

1. [Rigetti Forest Notebooks](https://gitlab.com/rigetti/jupyterhub/forest-notebook)
2. [Introduction to QAOA (PennyLane)](https://pennylane.ai/qml/demos/tutorial_qaoa_intro)
3. [MaxCut with QAOA](https://pennylane.ai/qml/demos/tutorial_qaoa_maxcut)
4. [IBM QAOA Tutorial](https://quantum.cloud.ibm.com/docs/en/tutorials/quantum-approximate-optimization-algorithm)
