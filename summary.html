Problem

Forecast a 14-tenor × 16-maturity swaption implied volatility surface (224 points) from 5-day historical windows. Only 489 training samples are available, making deep recurrent models unreliable. Primary metric: QLIKE — the financially correct volatility loss that penalises under-prediction asymmetrically.

Mode split: 5 input modes receive angle-encoded PCA features per timestep; 3 memory modes accumulate temporal state — photonic realisation of Li et al. (arXiv:2505.13933).

Feature vector: 5 timesteps × 8 Fock measurements = 40 quantum + 25 classical PCA = 65 hybrid features.

Fixed reservoir: circuit weights frozen after random init — no gradient computation, no barren plateaus, deterministic every run.

Temporal QRC Pivot

Ablation over α ∈ {0.0001…100}. α=0.1 unlocks quantum feature contribution.
